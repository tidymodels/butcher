# new_model_butcher

    Code
      new_model_butcher("potato", "Potato")
    Condition
      Error:
      ! `Potato` referenced is not installed.

---

    Code
      new_model_butcher("poe TAY toe", "Potato")
    Condition
      Error:
      ! `poe TAY toe` cannot have any spaces.

---

    Code
      new_model_butcher(1, "Potato")
    Condition
      Error:
      ! `1` must be a string.
